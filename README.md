# screenshot

Public screenshot hosting used for PR reviews and misc projects.

## Structure

- `pr-assets/<repo>/<pr-number>/...` — images referenced from PR comments
- `misc/<project>/...` — anything else

## Linking

Prefer `raw.githubusercontent.com` URLs in comments, e.g.

`https://raw.githubusercontent.com/axel-rookbot/screenshot/main/pr-assets/ex338/1412/password-blur.png`
